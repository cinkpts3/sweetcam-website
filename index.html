Implementation Plan

Project Overview

This project aims to transform SweetCam from a basic honeypot into a versatile, multi-protocol security tool.

Template-Based Device Emulation

  • YAML templates define metadata, protocols, and UI
  • Supports inheritance and JSON Schema validation
  • CLI tools: sweetcam template validate, list

Protocol Support

RTSP: Full RFC2326 support + RTP/RTCP packet simulation

ONVIF: WS-Discovery + Device Management, Media, PTZ services

Docker Enhancements

  • Multi-stage builds, health checks
  • Auto-config generation & Docker Compose profiles

HTTP Interface

  • Dynamic error simulation
  • i18n/multi-language support
  • Fake password reset flow with lockouts
