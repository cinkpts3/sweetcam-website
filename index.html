SweetCam GSoC 2025 Implementation Plan

Enhancing the IP Camera Honeypot with Multi-Protocol Support & Device Emulation

Project Overview

This project aims to transform SweetCam from a basic honeypot into a versatile, multi-protocol security tool...

Template-Based Device Emulation

  • YAML-based templates defining device metadata, protocols, and web interface features.
  • Inheritance support and JSON Schema validation.
  • CLI utilities like sweetcam template validate and sweetcam template list.

New Protocol Support

RTSP Enhancements

  • Implement full RTSP state machine (RFC 2326)
  • Methods: DESCRIBE, SETUP, PLAY, TEARDOWN
  • RTP/RTCP packet simulation (static images, video files, test patterns)

ONVIF Implementation

  • WS-Discovery responder for LAN discovery
  • Core services like Device Management, Media, PTZ control

Docker Usability Enhancements

  • Multi-stage Docker builds
  • Health checks, log rotation, and auto-config generation
  • Docker Compose profiles for various setups

HTTP Interface Improvements

  • Dynamic error generation (invalid login, session timeout)
  • Internationalization (i18n) with multiple language support
  • Password recovery flow (fake email/SMS, lockout mechanisms)
